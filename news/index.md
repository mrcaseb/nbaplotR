# Changelog

## nbaplotR 0.1.0

- Added support for logos in `gt` tables. The functions
  [`gt_nba_logos()`](https://mrcaseb.github.io/nbaplotR/reference/gt_nba_logos.md)
  and
  [`gt_wnba_logos()`](https://mrcaseb.github.io/nbaplotR/reference/gt_wnba_logos.md)
  render logos in gt table cells. The function
  [`gt_nba_cols_label()`](https://mrcaseb.github.io/nbaplotR/reference/gt_nba_cols_label.md)
  renders NBA/WNBA logos in gt table column labels.

## nbaplotR 0.0.4

- Allow invalid team names. `ggpath` (\>v1.0.1) will skip them and print
  a warning.
