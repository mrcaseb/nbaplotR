# nbaplotR: NBA Logo Plots in 'ggplot2' and 'gt'

A set of functions to visualize National Basketball League analysis in
'ggplot2'.

## Usage

``` r
GeomNBAlogo
```

## Format

An object of class `GeomNBAlogo` (inherits from `Geom`, `ggproto`, `gg`)
of length 5.
