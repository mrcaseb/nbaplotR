# Package index

## Geoms

ggplot2 geoms (geometric objects) that create a `layer()`. A layer
combines data, aesthetic mapping, a geom (geometric object), a stat
(statistical transformation), and a position adjustment.

- [`geom_nba_logos()`](https://mrcaseb.github.io/nbaplotR/reference/geom_nba_logos.md)
  : ggplot2 Layer for Visualizing NBA Team Logos
- [`geom_wnba_logos()`](https://mrcaseb.github.io/nbaplotR/reference/geom_wnba_logos.md)
  : ggplot2 Layer for Visualizing WNBA Team Logos

## Scales

Scales control the details of how data values are translated to visual
properties. Override the default scales to tweak details like the axis
labels or legend keys, or to use a completely different translation from
data to aesthetic.

- [`scale_color_nba()`](https://mrcaseb.github.io/nbaplotR/reference/scale_nba.md)
  [`scale_colour_nba()`](https://mrcaseb.github.io/nbaplotR/reference/scale_nba.md)
  [`scale_fill_nba()`](https://mrcaseb.github.io/nbaplotR/reference/scale_nba.md)
  [`scale_color_wnba()`](https://mrcaseb.github.io/nbaplotR/reference/scale_nba.md)
  [`scale_colour_wnba()`](https://mrcaseb.github.io/nbaplotR/reference/scale_nba.md)
  [`scale_fill_wnba()`](https://mrcaseb.github.io/nbaplotR/reference/scale_nba.md)
  : Scales for NBA & WNBA Team Colors

## Tables

Utility functions that work with the gt package to create html tables.

- [`gt_nba_cols_label()`](https://mrcaseb.github.io/nbaplotR/reference/gt_nba_cols_label.md)
  : Render Logos in 'gt' Table Column Labels
- [`gt_nba_logos()`](https://mrcaseb.github.io/nbaplotR/reference/gt_nba_logos.md)
  : Render NBA Logos in 'gt' Tables
- [`gt_wnba_logos()`](https://mrcaseb.github.io/nbaplotR/reference/gt_wnba_logos.md)
  : Render WNBA Logos in 'gt' Tables

## Theme Elements

Themes control the display of all non-data elements of the plot. Theme
elements can tweak individual settings inside `theme()`.

- [`element_nba_logo()`](https://mrcaseb.github.io/nbaplotR/reference/element.md)
  [`element_wnba_logo()`](https://mrcaseb.github.io/nbaplotR/reference/element.md)
  : Theme Elements for Image Grobs

## Utilities

Various helper functions.

- [`clean_team_abbrs()`](https://mrcaseb.github.io/nbaplotR/reference/clean_team_abbrs.md)
  : Standardize NBA/WNBA Team Abbreviations
- [`valid_team_names()`](https://mrcaseb.github.io/nbaplotR/reference/valid_team_names.md)
  : Output Valid NBA or WNBA Team Abbreviations
- [`nba_team_abbr_mapping`](https://mrcaseb.github.io/nbaplotR/reference/nba_team_abbr_mapping.md)
  : Alternate NBA Team Abbreviation Mapping
- [`wnba_team_abbr_mapping`](https://mrcaseb.github.io/nbaplotR/reference/wnba_team_abbr_mapping.md)
  : Alternate WNBA Team Abbreviation Mapping
